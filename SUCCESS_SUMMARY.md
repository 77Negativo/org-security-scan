O workflow é **100% genérico** e pode ser usado em qualquer organização GitHub:

1. **Copie** a pasta `org-security-scan`
2. **Crie** token: `repo`, `admin:org:read`, `read:user`
3. **Adicione** secret: `ORG_SECURITY_TOKEN`
4. **Execute** via GitHub Actions
